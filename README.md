# JavaScript Intern Projects Portfolio

Welcome to the JavaScript Intern Projects Portfolio! This repository contains 8 high-quality, perfectly crafted web applications demonstrating strong foundational web development skills.

Each project has been rigorously tested, completely styled with modern CSS, and refactored with robust Vanilla JavaScript logic. 

## 🚀 Tech Stack
- **HTML5**: Semantic and accessible markup.
- **CSS3**: Modern Flexbox/Grid, CSS Variables, and responsive design.
- **JavaScript (ES6+)**: Vanilla JS for DOM manipulation, state management, and external API integrations.
- **Local Storage**: For data persistence across sessions.

## 📂 Projects Overview

1. **Age Calculator**: Calculates exact age in years, months, and days with strict date validation.
2. **Rock Paper Scissors**: Classic game with a sleek dark-mode UI and comprehensive scoreboard.
3. **Random Quote Generator**: Fetches quotes via API, allows copying to clipboard, and features a local favorite vault.
4. **Image Search App**: Unsplash API integration with beautiful image grids, hover states, and pagination.
5. **Interactive Quiz**: 10-question Single Page Application (SPA) with a strict grading scale and detailed review screen.
6. **Password Generator**: Secure, customizable generator with strength calculation and a managed password vault.
7. **Stopwatch**: Precision `Date.now()` timer tracking milliseconds with robust chronological split lap logic.
8. **Task Manager (CRUD)**: Full task management app with inline editing, active/completed filtering, and smooth UI animations.

## 🛠 How to Run Locally

You do not need any build tools, Node.js, or package managers to run these projects. 

1. Clone this repository:
   ```bash
   git clone https://github.com/Krishypatel-4061/js-intern-projects.git
   ```
2. Navigate into the folder:
   ```bash
   cd js-intern-projects
   ```
3. Open `index.html` in your web browser to view the Portfolio Hub. From there, you can click on any card to navigate directly to the individual project.

Enjoy exploring the code!
